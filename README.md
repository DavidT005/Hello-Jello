# Hello Jello
### Description

This is a protoype for a game where you are a jelly trying yo survive.
On this game you can:


- Get burned by lava
- Get crushed by falling blocks
- ✨Ascend ✨  

### Features

- Main menu
- Game
- Extra big scores
- Press R to restart!
- Lava and blocks
- Bugs, the digital kind
